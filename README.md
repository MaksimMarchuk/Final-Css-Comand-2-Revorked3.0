# Team project "Ice Cream Cafe"

The website for the ice cream parlor is written in SCSS to develop teamwork skills and review what has been learned over the semester.


## Technologies Used
- HTML
- SCSS
- JavaScript

## Project goal

- Develop teamwork skills: task distribution, communication, working with Git.
---
- Consolidate knowledge of HTML, SCSS acquired during the semester.
---
- Learn to create a responsive website for different devices.
---
- Practice working with UI/UX
---
- Implement interactive elements using JavaScript.

## Structure
### 1. Home Page
---
**Section "Products"**  
When clicked, a modal window opens with ingredients, KBJU and a feedback form (Name, Phone, Comment).

**Section About**  
The "Read more" button activates a modal window with a video

**Section "Contacts"**  
The "Our Locations" and "Franchise" buttons activate the corresponding modal windows with the address and feedback

**Section "Reserve a table"**  
The "Reserve" button initiates the transition to the registration page.

### 2. Registration/Login Form
---
**Page "Create Account"**
- If the user enters the email or password incorrectly - an error is displayed
- After clicking the "Continue" button, the user is transferred to the Reservation page.
- If the user clicks on the "Log In" button, it redirects them to the login page

**"Log in" page**
- After clicking on the "Continue" button, the user goes to the Reservation page.
- If the user clicks on the "Sign In" button, it redirects them to the registration page.

### 3. Reservation page
---
- Clicking on the logo takes you to the main page. 
   
***Form:***
- Interactive drop-down calendar (activated by clicking on the calendar icon).
- Drop-down menu with a list of hours (activated by clicking on the clock icon).
- Drop-down list where you can select a table number (activated by clicking on the arrow)
